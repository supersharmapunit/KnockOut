Instruction

  1. Play in fullscreen for better experience
  2. All buttons inside the playground are diabled
  3. You have to press start button before every match in order to play the match
  4. Scores of current match are visible but the game will over when someone's total score will be 3
  5. White line indicates whose turn is it meanwhile other player's shoot button will not work
  6. Every player has 5 lives in a match and each Game will have 3 rounds
  7. Reset Match - It will reset both player's heath to 100 in current match
  8. Reset Game - It will reset the entire game including the main scores
